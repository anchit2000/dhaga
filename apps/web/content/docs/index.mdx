**Dhaga** (धागा — "thread") turns every card scan, badge, and voice note into a
private knowledge graph you can search in plain language. It is open-core: the
whole CRM is AGPL and runs fully self-hosted, with a source-available cloud
tier layered on top.

These docs cover running Dhaga yourself, extending it with your own providers,
the generated `@dhaga/core` API reference, and the product thinking behind it.
